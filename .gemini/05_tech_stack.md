# Pila Tecnológica (Tech Stack)

- **Lenguaje Principal:** Python 3.x
- **Librerías de Seguridad/Redes:** `dnspython`, `python-whois`, `scapy`, `socket`, `requests`, `concurrent.futures`.
- **Base de Datos Vectorial (RAG):** ChromaDB
- **Framework RAG:** LangChain
- **LLM (RAG):** Llama 3 a través de la API de Groq.
- **Documentación:** LaTeX (con paquetes como `babel`, `listings`, `graphicx`, `tikz`, `siunitx`).
- **Control de Versiones:** Git y GitHub.
